# CursorHaloOS

This repository contains the landing page for Cursor Halo, a context-aware AI assistant for macOS.

## Repository Structure

The main landing page is located at `index.html` in the root directory.

### Recent Changes

The original file structure had the HTML file nested in a `CursorHaloOG` folder. This has been restructured to place the landing page directly in the root directory as `index.html` for easier access and deployment.

## File History

- The original file was located at: `CursorHaloOG/cursor-halo-landing-original.html`
- It has been moved to: `index.html` (root directory)

This simplifies the repository structure and makes it easier to deploy the landing page to hosting services like GitHub Pages.
